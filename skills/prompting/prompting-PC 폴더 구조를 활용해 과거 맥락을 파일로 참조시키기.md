# PC 폴더 구조를 활용해 과거 맥락을 파일로 참조시키기

Claude Code의 핵심 장점은 PC 폴더 구조를 활용해 과거 히스토리와 맥락을 파일로 저장·참조시킬 수 있다는 점. 보고서 작성 시 과거 유사 주제 보고서, 관련 데이터를 함께 폴더에 넣고 지시하면 채팅창 대비 비교 불가 수준의 초안이 출력된다.

---

## Usage

Claude Code에서 보고서를 작성할 때, 프로젝트 폴더 내 과거 유사 보고서와 관련 데이터 파일을 함께 배치한 뒤 '이 폴더의 기존 보고서 스타일과 데이터를 참고해서 [주제]에 대한 보고서 초안을 작성해줘'라고 지시하라. 맥락 파일이 많을수록 출력 품질이 비약적으로 올라간다.
