# Hook으로 플랜 검증 및 작업 완료 후 자동 검토 구현하기

Claude Code가 제공하는 훅(PostToolUse 등)을 활용하여 현재 플랜에 추가 정보가 필요한지 자동 점검하고, 진행한 작업이 제대로 수행되었는지 자동 검토하게 한다. 이는 Claude가 '작업 완료'라고 보고하지만 실제로는 누락된 절차가 있는 문제를 방지한다.

---

## Usage

# auto-review-hook: 작업 완료 후 자동 검토 훅

## 트리거
Claude Code의 PostToolUse 훅으로 작업 완료 시점에 자동 실행

## 절차
1. 작업 완료 시 훅이 트리거되면 현재까지의 작업 내역을 요약한다.
2. 원래 플랜 대비 누락된 절차가 없는지 점검한다.
3. 누락 사항이 발견되면 사용자에게 알리고 추가 작업을 제안한다.
4. 플랜 단계에서는 추가 정보가 필요한지 자동으로 확인한다.

## Gotchas
- 훅이 매 도구 호출마다 실행되므로 성능 영향을 고려해 검토 범위를 적절히 제한한다.
