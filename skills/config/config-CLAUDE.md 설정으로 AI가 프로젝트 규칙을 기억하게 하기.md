## Usage

# CLAUDE.md 설정으로 AI가 프로젝트 규칙을 지속 기억하게 만들기

## 트리거
사용자가 새 프로젝트를 시작하거나 CLAUDE.md 설정을 요청할 때

## 절차
1. 프로젝트 루트에 CLAUDE.md 파일 생성
2. 프로젝트 목적, 디렉터리 구조, 코딩 규칙, 주의사항을 섹션별로 작성
3. 팀 컨벤션(네이밍, 커밋 메시지 형식 등) 명시
4. 금지 사항(커밋 금지 파일, 보안 관련)을 별도 섹션으로 분리

## Gotchas
- CLAUDE.md가 너무 길면 컨텍스트를 낭비하므로 핵심만 간결하게 유지
- .env, 쿠키 파일 등 민감 정보는 절대 CLAUDE.md에 포함하지 말 것
