# CLAUDE.md에 작업 원칙과 규칙을 명시해 반복 실수 방지

CLAUDE.md 파일에 '무엇을 하는 프로젝트인가'보다 '어떤 원칙으로 작업하는가'를 기록한다. 코드보다 설계 우선, 한국어 주석 표준, 특정 폴더 수정 금지 같은 규칙을 명시하면 Claude가 매번 같은 실수를 반복하지 않는다.

---

## Usage

# CLAUDE.md 작업 원칙 템플릿 생성기

사용자의 프로젝트 정보를 받아 CLAUDE.md에 들어갈 작업 원칙 섹션을 생성한다.

## 입력
- 프로젝트 유형 (웹앱, CLI, 라이브러리 등)
- 사용 언어/프레임워크
- 팀 컨벤션 (있으면)

## 출력 포맷
```
## 코딩 규칙
- [원칙 1]
- [원칙 2]

## 금지 사항
- [금지 1]

## 우선순위
- [우선순위 1]
```
